3.2 docker

安装docker 环境,centos/ubuntu

  • yum -y install docker-io 安装docker

  • /etc/docker/daemon.json "registry-mirrors": ["http://hub-mirror.c.163.com"] 修改镜像加速

  • 适用于 CentOS 7 版本。

  • 修改 Docker 配置文件 vi /etc/sysconfig/docker，如下：
  • OPTIONS='--registry-mirror=https://mirror.ccs.tencentyun.com'

Docker的容器创建以及基本命令

  1. 需要将8.8.8.8和8.8.4.4的DNS服务器IP添加到域名系统中去。

    • Ubuntu下的操作是打开/etc/resolv.conf，添加两行nameserver 8.8.8.8和nameserver 8.8.4.4。

  2. 接下来使用命令

    • docker run -it con_name /bin/bash 创建一个容器并进入
    • sudo docker run -it --name=felix centos 命名

  3. 容器常用的管理命令

    • docker run -d 创建一个守护式容器在后台运行。
    • docker ps -a 查看已经创建的容器
    • docker ps -s 查看已经启动的容器
    • docker start con_name 启动容器名为con_name的容器
    • docker stop con_name 停止容器名为con_name的容器
    • docker rm con_name 删除容器名为con_name的容器
    • docker rename o_name n_name 重命名一个容器
    • docker attach con_name 唤醒
    • docker logs con_name 获取容器名为con_name的容器日志
    • docker inspect 查看容器的详细信息
    • docker top con_name 查看容器名为con_name的容器内部的进程
    • docker exec -it mysql bash 进入容器
