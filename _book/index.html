git 指引

  1. 重新克隆一次项目 git clone github.com
  2. 克隆路径为 /new/project,本地修改路径为 /old/project。
  3. 进入旧目录 cd /old/project。
  4. 同步 rsync -av ./ /new/project/
  5. 回到新项目中，做添加和提交的操作。
